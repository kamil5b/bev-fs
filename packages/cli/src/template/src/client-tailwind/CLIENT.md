# Client Documentation

This is the client-side application built with Vue 3, Vite, and Tailwind CSS.

## Structure

- `components/` - Reusable Vue components
- `composables/` - Vue composables for logic reuse
- `pages/` - Page components for routing
- `router/` - Route definitions
- `App.vue` - Root component
- `main.ts` - Application entry point
- `index.html` - HTML template

## Features

- **Responsive Design** - Tailwind CSS for responsive, mobile-first styling
- **Modern UI Components** - Pre-built components with Tailwind styling
- **Type-Safe** - Full TypeScript support
- **API Integration** - Composable for easy API communication

## Available Scripts

- `bun run dev:client` - Start Vite dev server
- `bun run build:client` - Build for production

## Styling

This template uses **Tailwind CSS** for all styling. No custom CSS files are needed for component styling - all styles are defined using Tailwind's utility classes.

### Key Tailwind Features Used

- Responsive design with responsive prefixes (`sm:`, `md:`, `lg:`)
- Dark mode support with `dark:` prefix
- Custom color palette for the design
- Component classes defined in `index.css`

## Components

All components use Tailwind CSS for styling and follow a consistent design system.
